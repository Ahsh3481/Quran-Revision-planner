#  Quran Revision Planner

A simple web based planner that generates a structured Quran revision schedule based on selected Surahs, date range, and reading pace. It also allows exporting the plan as a calendar file (`.ics`) for use in Google Calendar, Apple Calendar, or other calendar apps.

The website is HTTPS;//Quranrevision.co.uk


---

##  Features

- Select **start and end Surah**
- Choose a **start and end date**
- Automatically generates a **daily revision plan**
- Splits reading into **pages per day based on total duration**
- Maps pages back to corresponding Surahs
- Export plan as a **calendar file (.ics)** with reminders
- Works fully in the browser (no backend required)

---

##  How It Works

1. User selects a start and end Surah.
2. User selects a date range.
3. The app:
   - Calculates total pages between Surahs
   - Divides pages evenly across selected days
   - Generates a day-by-day reading plan
4. Each day is converted into a calendar event for export.
